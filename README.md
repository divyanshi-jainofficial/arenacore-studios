# ArenaCore Studios 🎮

A modern gaming tournament website built using HTML, CSS, and JavaScript.

## Features
- Animated gaming UI
- Theme switcher
- Responsive design
- 3D hover effects
- Tournament showcase
- Registration form
- Video background

## Tech Stack
- HTML5
- CSS3
- JavaScript
